Making a game with Fusion 2.5

by lime360

Introduction

Clickteam Fusion 2.5 (or simply Fusion 2.5) is a software that allows people to build their own game WITHOUT CODING.

While the other software with similar functionality exists (Game Maker: Studio 2, Construct 3, Unity), Fusion 2.5 is the most used game development software that is best suitable for beginner coders or non-coders.

Clickteam offers these export modules to publish games everywhere:

 It’s bulit-in Windows export module

HTML5 export module for creating browser games

Flash export module (ADOBE FLASH IS DEAD!!!!!!1!!!1!!11!!)

Android export module for creating games for your Android phone

UWP export module for creating modern Windows 10/11 apps

XNA export module for creating Xbox 360 games (CANCELED)

 iOS export module for creating games for an iPhone

Mac export module for bringing your creations to the Mac

Sounds cool! I need it now!

Download the free version that comes with a limited HTML5 exporter. It doesn’t take a while to download!

When you’re done, click on the installation file named cf25freeSetup.exe

Then you do anything like you usually do, agree to the license agreement, click random buttons, and you're done!

Let’s fancy it up with events!

As I said before, Fusion 2.5 has Event Editor, meaning you can make games using visual scripting.

To do that, click on an Event Editor icon  or Event List Editor icon .

These icons above are bulit-in objects in every Fusion 2.5 game:

 Special - used for creating loops, working with global variables.

 Sound - used for adding music and sounds.

 Storyboard Controls - used for changing frames.

 The timer - used for conditions such as: “Every 0,30 seconds” and “If the timer is equal 10 seconds”.

 Create new objects - used for creating new objects or copying the same object.

 The mouse pointer and keyboard - used for making events with mouse and keyboard.

 Player - used for changing scores and lives and making events for another player.
